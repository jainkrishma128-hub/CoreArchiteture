## CSS Frameworks & Libraries

### 1. Bootstrap 5

**Location**: `wwwroot/lib/bootstrap/`

**CDN**: Local files (downloaded)

**Version**: Bootstrap 5.x (based on file structure)

**Purpose**: Primary CSS framework for responsive layout, components, and utilities

**Included Files:**
- `bootstrap.min.css` - Main Bootstrap CSS
- `bootstrap.css.map` - Source map for debugging
- `bootstrap-grid.min.css` - Grid system only
- `bootstrap-utilities.min.css` - Utility classes only
- `bootstrap-reboot.min.css` - Normalize and reset styles

**JavaScript Bundle:**
- `bootstrap.bundle.min.js` - Includes Popper.js (for tooltips, popovers, dropdowns)

**Usage:**
```html
<!-- In Layout Files -->
<link rel="stylesheet" href="~/lib/bootstrap/dist/css/bootstrap.min.css" />
<script src="~/lib/bootstrap/dist/js/bootstrap.bundle.min.js"></script>
```

**Features Used:**
- Grid system (container, row, col-*)
- Navigation bars
- Buttons, forms, cards
- Modals, dropdowns, tooltips
- Responsive utilities
- Bootstrap 5 components (alerts, badges, breadcrumbs, etc.)

## NuGet Packages (UI Related)

### 1. NToastNotify

**NuGet Package**: `NToastNotify` Version 8.0.0

**Purpose**: Toast notification library for ASP.NET Core

**Features:**
- Success, error, warning, info notifications
- Toastr.js integration
- Multiple notification themes
- Auto-dismiss notifications

**Configuration** (Program.cs):
```csharp
builder.Services.AddControllersWithViews().AddNToastNotifyNoty(new NotyOptions
{
    ProgressBar = true,
    Timeout = 5000,
    Theme = "metroui"
});
```

**Usage in Controllers:**
```csharp
_toastNotification.AddSuccessToastMessage("Product created successfully");
_toastNotification.AddErrorToastMessage("Error creating product");
```

**Usage in Views:**
```html
@await Component.InvokeAsync("NToastNotify")
```

## Usage Examples

### Complete Layout Example (Admin)

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>@ViewData["Title"] - Admin Panel</title>
    
    <!-- CSS Libraries -->
    <link rel="stylesheet" href="~/lib/bootstrap/dist/css/bootstrap.min.css" />
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css" />
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css" />
    
    <!-- Custom CSS -->
    <link rel="stylesheet" href="~/css/site.css" asp-append-version="true" />
    <link rel="stylesheet" href="~/css/admin.css" asp-append-version="true" />
</head>
<body class="admin-body">
    <!-- Page Content -->
    
    <!-- JavaScript Libraries -->
    <script src="~/lib/jquery/dist/jquery.min.js"></script>
    <script src="~/lib/bootstrap/dist/js/bootstrap.bundle.min.js"></script>
    
    <!-- Custom JavaScript -->
    <script src="~/js/site.js" asp-append-version="true"></script>
    <script src="~/js/admin.js" asp-append-version="true"></script>
    
    <!-- Toast Notifications -->
    @await Component.InvokeAsync("NToastNotify")
    
    @await RenderSectionAsync("Scripts", required: false)
</body>
</html>
```

### Form Validation Example

```html
@model CreateProductDto

<form asp-action="Create" method="post" id="createForm">
    <div class="form-group mb-3">
        <label asp-for="Name" class="form-label"></label>
        <input asp-for="Name" class="form-control" />
        <span asp-validation-for="Name" class="text-danger"></span>
    </div>
    
    <button type="submit" class="btn btn-primary">Create</button>
</form>

@section Scripts {
    @{await Html.RenderPartialAsync("_ValidationScriptsPartial");}
    
    <script>
        $(document).ready(function() {
            $('#createForm').validate({
                rules: {
                    Name: {
                        required: true,
                        minlength: 3
                    }
                }
            });
        });
    </script>
}
```

### AJAX Request Example

```javascript
$(document).ready(function() {
    // Load products via AJAX
    $.ajax({
        url: '/Admin/Products/GetAll',
        method: 'GET',
        success: function(response) {
            if (response.success) {
                populateTable(response.data);
            }
        },
        error: function(xhr, status, error) {
            console.error('Error loading products:', error);
        }
    });
});
```
